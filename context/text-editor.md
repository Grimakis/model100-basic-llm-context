# Text Editor (TEXT) Quick Reference

## Entering the Text Editor program
Position the Menu Cursor over `TEXT` or a `.DO` text file and press ENTER; text files open in the Text Editor automatically.

## Cursor Control Keys
- Right Arrow: Moves the Cursor one position to the right.
- Left Arrow: Moves the Cursor one position to the left.
- Up Arrow: Moves the Cursor one position up.
- Down Arrow: Moves the Cursor one position down.
- CTRL + Right Arrow: Moves the Cursor to the right end of the line.
- CTRL + Left Arrow: Moves the Cursor to the left end of the line.
- CTRL + Up Arrow: Moves the Cursor to the top of the file.
- CTRL + Down Arrow: Moves the Cursor to the bottom of the file.
- SHIFT + Right Arrow: Moves the Cursor to the beginning of the word to immediate right.
- SHIFT + Left Arrow: Moves the Cursor to the beginning of the word to immediate left.
- SHIFT + Up Arrow: Moves the Cursor to the top of the screen above the current position.
- SHIFT + Down Arrow: Moves the Cursor to the bottom of the screen below the current position.

## Text Manipulation Keys
- `DEL`: Delete the character at the current Cursor position.
- `BKSP`: Delete the character to the left of the current Cursor position.
- `F1 Find`: Search the file for a string; TEXT prompts for the match.
- `F2 Load`: Read an ASCII data file into RAM; TEXT prompts for the filename.
- `F3 Save`: Write a data file; TEXT prompts for the filename.
- `F5 Copy`: Duplicate the selected text into the paste buffer.
- `F6 Cut`: Move the selected text from the screen into the paste buffer.
- `F7 Select`: Start defining text for copying or deleting.

## Special Commands and Keys
- `FB`: Exit the Text Editor and return to the Menu.
- `Menu`: Show the function-key list.
- `(PASTE)`: Insert the paste buffer at the current cursor.
- `LABEL`: Print the function-key definitions on the bottom display line.
- `(PRINT)`: Print the screen contents on the printer.
- `SHIFT (PRINT)`: Print the file contents (TEXT prompts for printer width).
- `(TAB)`: Insert an eight-character tab.
- `(CTRL A)`: Move the cursor to the start of the previous word.
- `CTAL B`: Move the cursor directly to the bottom of the screen.
- `CTRL C`: Cancel Select, Save, Load, Find, or Print.
- `CTROD`: Move the cursor one character to the right.
- `CTRL CE`: Move the cursor up one line.
- `(CTRL F)`: Move the cursor to the beginning of the next word.
- `(CTRL G)`: Save a data file to cassette tape.
- `CTRL CH`: Delete the previous character.
- `CTRL I`: Insert an eight-character tab.
- `CTALOL`: Enter Select Text mode.
- `GTALOND`: Insert a carriage return and line feed.
- `CTALON`: Find a text string.
- `CTRLO`: Copy a text string.
- `(CTRL P)`: Save the next keystroke as a non-printing control character.
- `CTAL Q`: Move the cursor to the left end of the current line.
- `(CTRL R)`: Move the cursor to the right end of the line.
- `(CTRL S)`: Move the cursor one character to the left.
- `CTALICT`: Move the cursor to the top of the screen directly above.
- `CTALCU`: Move the selected string from the screen to the paste buffer.
- `CTALEV`: Load a data file from cassette tape.
- `CTRLOW`: Move the cursor to the beginning of the file.
- `CTRL??`: Move the cursor down one line.
- `CTALY`: Print the entire file.
- `(CTRL Z)`: Move the cursor to the end of the file.
